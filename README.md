# Harvest Tweaks

A mod for **The Long Dark** that lets you adjust harvesting time for branches and limbs via sliders in the Mod Settings menu.

## Features

- Adjust harvest time for each object individually:
  - Branch
  - Cedar Limb / Fir Limb
  - Big Cedar Limb / Big Fir Limb
- Fully localized in all 18 languages supported by The Long Dark
- Settings saved automatically between sessions

## Vanilla Values

| Object        | Time   |
|---------------|--------|
| Branch        | 0.17 h |
| Limbs         | 1.50 h |
| Big Limbs     | 1.50 h |

## Requirements

- [MelonLoader](https://melonwiki.xyz/) 0.6.x
- [ModSettings](https://www.nexusmods.com/thelongdark/mods/111) 2.2.5+

## Installation

1. Install MelonLoader
2. Install ModSettings
3. Drop `HarvestTweaks.dll` into your `Mods/` folder
4. Launch the game
